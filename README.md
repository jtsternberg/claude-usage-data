# Claude Code Usage Insights

This repository contains usage analytics and insights generated from Claude Code sessions.

## What is this?

The `index.html` file is an interactive dashboard generated by Claude Code's `/insights` command. It provides a comprehensive analysis of your Claude Code usage patterns including:

- **Session statistics** - Total sessions, conversations, and token usage
- **Cost analytics** - Spending breakdown by model and time period
- **Project areas** - Which projects you've worked on most
- **Tool usage** - Which Claude Code tools are used most frequently
- **Friction points** - Common issues and areas for improvement
- **Productivity insights** - Patterns in how you use Claude Code effectively

## Viewing the report

The report is hosted via GitHub Pages at:
https://jtsternberg.github.io/claude-usage-data/

Or open `index.html` locally in any web browser.

## Publish Your Own

Want to publish your own Claude Code usage insights to GitHub Pages? Here's how:

### 1. Generate Your Report

In any Claude Code session, run:

```
/insights
```

### 2. Install the Publish Skill

Run this one-liner to install the `/publish-insights` skill:

```bash
mkdir -p ~/.claude/skills/publish-insights && curl -sL https://raw.githubusercontent.com/jtsternberg/claude-usage-data/main/skills/publish-insights/SKILL.md -o ~/.claude/skills/publish-insights/SKILL.md
```

### 3. Publish to GitHub Pages

In Claude Code, run:

```
/publish-insights
```

This will guide you through creating a GitHub repo and publishing your report to GitHub Pages.
